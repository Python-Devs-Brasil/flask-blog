# Flask-Blog 

Modules & *FrameWork's*:
* Flask
* Flask-Migrate
* Flask-SQLAlchemy
* Flask-Markdown
* Markdown

# Getting Started 

1. `` python manage_flask.py db init ``
2. `` python manage_flask.py db migrate ``
3. `` python manage_flask.py db upgrade ``
4. `` python start.py ``
5. `` python server.py ``
6. Open in your browser http://0.0.0.0:4242/
