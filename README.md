1. `` python manage_flask.py db init ``
2. `` python manage_flask.py db init ``

4. `` python server.py ``
5. Open in your browser http://0.0.0.0:4242/


Obs: if you need a modules or framework's, run in terminal `` pip install -r requirements.txt ``
